The Mutation type represents all of the mutable entry points into the API.

## Fields

### locationCreate ([Location](./Location))
Create new location.
Authorization: must be at least ADMIN.
Returns created location.

| ARGUMENT  | TYPE    | NULLABLE | LIST | DESCRIPTION |
| - | - | - | - | - |
| config | [BatchLocationInput](../ecdn-api-objects/BatchLocationInput) | NO | NO | Batch input for location creation. |

### locationConfigure ([Location](./Location))
Batch configuration of a location.
Authorization: must be at least ADMIN.
Returns configured location.

| ARGUMENT  | TYPE    | NULLABLE | LIST | DESCRIPTION |
| - | - | - | - | - |
| locationID | [ID](../ecdn-api-objects/ID) | NO | NO | ID of location to configure. |
| config | [BatchLocationInput](../ecdn-api-objects/BatchLocationInput) | NO | NO | Batch input for location configuration. |

### locationSetParents ([Void](./Void))
Set a common parent for locations. Parent can either be a region or a location.

| ARGUMENT  | TYPE    | NULLABLE | LIST | DESCRIPTION |
| - | - | - | - | - |
| locationIDs | [ID](../ecdn-api-scalars/ID) | NO | YES | IDs of (children) locations (which we set a parent for). |
| parentID | [ID](../ecdn-api-scalars/ID) | NO | NO | ID of parent location/region. |

### reportCreate ([Report](./Report))
Create time-series report of ECDN video delivery.
Interval of a query with minutely granularity cannot exceed 24 hours.
Metrics older than 1 year are discarded.
Returns created report.

| ARGUMENT  | TYPE    | NULLABLE | LIST | DESCRIPTION |
| - | - | - | - | - |
| granularity | [Granularity](../ecdn-api-objects/Granularity) | NO | NO | Granularity of time intervals. |
| breakdown | [ReportBreakdown](../ecdn-api-objects/ReportBreakdown) | YES | NO | Break down report's intervals by additional fields. |
| from | [Time](../ecdn-api-objects/Time) | NO | NO | Start report from this point in time. |
| to | [Time](../ecdn-api-objects/Time) | NO | NO | End report at this point in time. |
| email | [EmailAddress](../ecdn-api-objects/EmailAddress) | NO | NO | Send created report to this email address. |
| include | [ReportField](../ecdn-api-objects/ReportField) | NO | YES | Included fields in the report. |

type Mutation {

	# Disable content delivery on servers.

	# Authorization: must be at least ADMIN.

	serverDisableRemoteAssistance(serverIDs: [ID!]): Void

}
