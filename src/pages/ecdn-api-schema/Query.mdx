The Query type represents all of the queryable entry points into the API.

```graphql
type Query {

	# Get all regions.
	regions: [Region!]!

	# Get all servers.
	servers: [Server!]!

	# Get server by ID.
	serverByID(
		# ID of queried server.
		id: ID!
	): Server!

	# Get all locations.
	locations: [Location!]!

	# Get location by ID.
	locationByID(
		# ID of queried location.
		id: ID!
	): Location!

	# Get logged in user.
	me: User!

}
```
