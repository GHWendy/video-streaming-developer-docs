## Get current background color

This request returns the color of the channel page background.

```
GET https://api.video.ibm.com/channels/{channel_id}/design/background.json
```

### Parameters

This request has no parameters.

### Success response

On success a response with HTTP status "200 OK" is returned with a key-value pair under a background element.

| KEY     | TYPE   | DESCRIPTION                                     |
| ------- | ------ | ----------------------------------------------- |
| `color` | string | Channel page background color in hex RGB format |

### Error responses

Possible error responses:

| HTTP RESPONSE CODE      | ERROR VALUE | ERROR CONDITIONS                                                                        |
| ----------------------- | ----------- | --------------------------------------------------------------------------------------- |
| 401 Unauthorized        |             | The provided access token is missing, revoked, expired or malformed                     |
| 404 Not Found           | `not_found` | Channel not found                                                                       |
| 503 Service Unavailable |             | There is a temporary error on the server which makes it impossible to serve the request |

## Set background color

Change the background color of the channel page.

```
PUT https://api.video.ibm.com/channels/{channel_id}/design/background.json
```

### Parameters

The parameters for the POST request:

| PARAMETER | TYPE   | IMPORTANCE | DESCRIPTION                                                    |
| --------- | ------ | ---------- | -------------------------------------------------------------- |
| `color`   | string | REQUIRED   | Channel page background color in hex RGB format (e.g. #FFFFFF) |

### Success response

On success a response with HTTP status “200 OK” is returned.

### Error responses

Possible error responses:

| HTTP RESPONSE CODE      | ERROR VALUE         | ERROR CONDITIONS                                                                        |
| ----------------------- | ------------------- | --------------------------------------------------------------------------------------- |
| 400 Bad Request         | `bad_request`       | Validation failed                                                                       |
| 401 Unauthorized        |                     | The provided access token is missing, revoked, expired or malformed                     |
| 403 Forbidden           | `lack_of_ownership` | The API user is not allowed to manage the given channel                                 |
| 404 Not Found           | `not_found`         | Channel not found                                                                       |
| 503 Service Unavailable |                     | There is a temporary error on the server which makes it impossible to serve the request |
