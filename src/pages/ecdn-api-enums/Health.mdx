Indicates health of given entity.

### Values

| NAME | DESCRIPTION |
| - | - |
| GOOD | Everything is working perfectly. |
| WARNING | Something isn't working perfectly but given service should still function. |
| POOR | Something isn't working which might cause given service to malfunction. |
