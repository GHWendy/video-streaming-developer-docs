Method of applying configuration to an ECDN server.

### Values

| NAME | DESCRIPTION |
| - | - |
| MANUAL | Manually configurate the ECDN server. A configuration CD must be mounted and the server must be restarted. |
| REMOTE | Remotely configure the ECDN server. ECDN server will pick up the desired changes once it checks in to the API. |
	
```graphql

	# Manually configurate the ECDN server.
	# A configuration CD must be mounted and the server must be restarted.
	MANUAL
	# Remotely configure the ECDN server.
	# ECDN server will pick up the desired changes once it checks in to the API.
	REMOTE

}
```
