Represents update status of an ECDN server.

```graphql
type Update {

	# ECDN software version on server. Null if server never checked in before.
	ecdnVersion: SemanticVersion

	# Current security patch level of server. Null if the server never checked in before.
	patchLevel: PatchLevel

}
```

### Fields

| NAME | TYPE | NULLABLE | LIST | DESCRIPTION |
| - | - | - | - | - |
| ecdnVersion | [SemanticVersion](../ecdn-api-scalars/SemanticVersion) | YES | NO | ECDN software version on server. Null if server never checked in before. | 
| patchLevel | [PatchLevel](../ecdn-api-scalars/PatchLevel) | YES | NO | Current security patch level of server. Null if the server never checked in before. |
